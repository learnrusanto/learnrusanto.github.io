# Произноша

В Русанто писмо, один знако отвечем всегда до точне один звуко.

"о" и "е" всегда јесем произноша.

Голоса звукоф (IPA)

|        |  Front  | Central | Back  |
|:------:|:-------:|:-------:|:-----:|
| Close  |   i     |         |   u   |
|  Mid   |   e     |         |   o   |
|  Open  |         |  a      |       |
